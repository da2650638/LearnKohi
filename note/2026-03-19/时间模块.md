# 三、正确做法：REALTIME + 格式化

---

## 🔧 标准实现（Linux）

```c
#include <time.h>
#include <stdio.h>

void platform_get_datetime(char* buffer, size_t size) {
    struct timespec ts;
    clock_gettime(CLOCK_REALTIME, &ts);

    struct tm tm_time;
    localtime_r(&ts.tv_sec, &tm_time);

    strftime(buffer, size, "%Y-%m-%d %H:%M:%S", &tm_time);
}
```

---

## 使用：

```c
char buf[64];
platform_get_datetime(buf, sizeof(buf));

printf("%s\n", buf);
```

👉 输出：

```text
2026-03-19 14:23:01
```

# 四、升级版（带毫秒/微秒，推荐你用）

你做引擎日志，强烈建议带 ms：

---

```c
void platform_get_datetime_ms(char* buffer, size_t size) {
    struct timespec ts;
    clock_gettime(CLOCK_REALTIME, &ts);

    struct tm tm_time;
    localtime_r(&ts.tv_sec, &tm_time);

    int ms = ts.tv_nsec / 1000000;

    snprintf(buffer, size,
        "%04d-%02d-%02d %02d:%02d:%02d.%03d",
        tm_time.tm_year + 1900,
        tm_time.tm_mon + 1,
        tm_time.tm_mday,
        tm_time.tm_hour,
        tm_time.tm_min,
        tm_time.tm_sec,
        ms
    );
}
```

---

## 输出：

```text
2026-03-19 14:23:01.123
```

👉 这个才是**工程日志标准格式**

#include "platform_time.h"
#include <time.h>
#include <stdio.h>

double platform_time_now() {
    struct timespec ts;
    clock_gettime(CLOCK_MONOTONIC, &ts);
    return ts.tv_sec + ts.tv_nsec * 1e-9;
}

uint64_t platform_time_ns() {
    struct timespec ts;
    clock_gettime(CLOCK_MONOTONIC, &ts);
    return (uint64_t)ts.tv_sec * 1000000000ULL + ts.tv_nsec;
}

void platform_time_string(char* buffer, uint32_t size) {
    struct timespec ts;
    clock_gettime(CLOCK_REALTIME, &ts);

    struct tm tm_time;
    localtime_r(&ts.tv_sec, &tm_time);

    strftime(buffer, size, "%Y-%m-%d %H:%M:%S", &tm_time);
}

void platform_time_string_ms(char* buffer, uint32_t size) {
    struct timespec ts;
    clock_gettime(CLOCK_REALTIME, &ts);

    struct tm tm_time;
    localtime_r(&ts.tv_sec, &tm_time);

    int ms = ts.tv_nsec / 1000000;

    snprintf(buffer, size,
        "%04d-%02d-%02d %02d:%02d:%02d.%03d",
        tm_time.tm_year + 1900,
        tm_time.tm_mon + 1,
        tm_time.tm_mday,
        tm_time.tm_hour,
        tm_time.tm_min,
        tm_time.tm_sec,
        ms
    );
}
